# Library-app 

- 도서 관리 API를 개발해보려고 한다. Library-app 프로젝트는 entity 연관관계 설계 연습 및 API 개발 연습이 목적이다.
- 주요 기능으로는 사용자는 책을 대출 및 반남할 수 있고, 책에 대해서 리뷰를 작성할 수 있다.




## API 스펙

| 기능       | HTTP Method | HTTP Path |
|----------|-------------|-----------|
| [사용자] 등록 | POST        | /user     |
| [도서] 등록  | POST        | /book     |
